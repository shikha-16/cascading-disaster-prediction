What is a cascade?

The definition Formal

A cascade is a primary weather event that causally triggers a secondary event. We identify cascades with five simultaneous constraints, grounded in peer-reviewed literature and NOAA guidance.
1
Temporal window — within 7 days
The secondary event must start no more than 7 days after the primary ends.
2
Same county (FIPS match)
Both events must occur in the same county, ensuring spatial proximity.
3
Shared NOAA episode ID
Our strongest filter — NWS forecasters grouped the events into one weather episode.
4
Different event types
Primary ≠ secondary, otherwise it's a continuation, not a cascade.
5
Validated causal pair
The (primary → secondary) pair must appear in our library of 100+ documented causal relationships.
